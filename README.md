# Rock Paper Scissors Game

## Overview

Rock Paper Scissors is an interactive browser-based game developed using HTML, CSS, and JavaScript. Players compete against the computer by selecting Rock, Paper, or Scissors. The game automatically determines the winner, updates scores, and displays the result after each round.

This project was created as part of my web development learning journey and was inspired by tutorials from Apna College. I implemented the game logic, score tracking, and user interaction using JavaScript.

## Features

* Interactive Rock, Paper, and Scissors selection
* Computer-generated random moves
* Real-time score tracking
* Win, lose, and draw detection
* Dynamic result messages
* Responsive and user-friendly interface
* Visual feedback with color changes

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)


## How the Game Works

1. The user selects Rock, Paper, or Scissors.
2. The computer randomly generates a choice.
3. The game compares both choices.
4. The winner is determined according to the rules:

   * Rock beats Scissors
   * Scissors beats Paper
   * Paper beats Rock
5. Scores are updated automatically.
6. The result is displayed on the screen.
